# Module 9: GlobalTech Sales Analysis

This repository contains my work for Module 9, focused on introductory data analysis with Pandas.

## Assignment Overview

In this assignment, I analyzed quarterly sales data for GlobalTech. The dataset included sales by region, store, category, product, unit price, total sales, and promotion status.

## What I Practiced

- Loading data into a Pandas DataFrame
- Exploring dataset structure with `.head()`, `.info()`, `.shape`, and `.describe()`
- Selecting columns and filtering rows
- Calculating total units, total revenue, and average unit price
- Grouping data by product, region, and category
- Comparing promoted and non-promoted sales
- Identifying missing values and missing value percentages
